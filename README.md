# Responsive Frontend Interface

A mobile-first, framework-free web interface built with semantic HTML5, CSS Grid + Flexbox, and vanilla JavaScript. 
Features a fluid responsive layout (clamp() typography, breakpoints at 768px/1024px), an accessible mobile nav with 
keyboard support, scroll-based active-link highlighting, and a live-validated contact form — all styled with a 
warm, grounded 2025 color palette (Mocha Mousse, Ethereal Blue, Moonlit Grey).

## Features
- Semantic landmarks (`header`, `nav`, `main`, `article`, `aside`, `footer`) for accessibility & SEO
- Mobile-first responsive design (no media query above min-width: 1024px needed for desktop)
- CSS Grid for macro layout, Flexbox for components
- Vanilla JS: hamburger nav state, scroll spy, back-to-top, form validation
- WCAG-conscious: skip link, aria-live regions, focus states, reduced-motion support

## Tech Stack
HTML5 · CSS3 (Grid, Flexbox, clamp()) · Vanilla JavaScript — no frameworks or libraries

## Getting Started
Clone the repo and open `index.html` in your browser — no build step required.
